# haskell-playground

> Playing with Haskell

## First steps

1. Install Haskell Platform: https://www.haskell.org/platform
- To start `repl`, just run `ghci`.
- To load a file inside `repl`, type `:l <yourfile.hs>` (`.hs` is a haskell extension)
